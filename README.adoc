= Ролевод

== Как нанести пользу

. Завести тикет и описать идею доработки.
. Настроить одну из link:envs/dev[поддерживаемых] сред разработки и зафиксировать её в `bakers/ansible/group_vars/all.yaml`.
+
[source,yaml]
----
# for example
devenv: church
----
. Выполнить доработку на <<ветки,отдельной>> ветке.
. Добиться успешного прохождения проверок на локальной машине.
+
[source,bash]
----
ansible-playbook codebase.yaml
ansible-playbook stacks.yaml
----
. Создать пул-реквест.
. Добиться успешного прохождения проверок в рамках CI.

== Ветки

=== feature/<concise-name>

Функциональность, которая открывает новые возможности для потенциальных пользователей.

=== bugfix/<concise-name>

Правка ошибок, опечаток и пр.

=== chore/<concise-name>

Рефакторинг, технический долг, мелкие улучшения и пр.
